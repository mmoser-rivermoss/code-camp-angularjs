What comes with AngularJS

AngularJS has a lot of pieces and parts. It is a very mature framework that is now supported by Google. There are some key components that make it a desired framework to use.

  • Dependency Injection
  • Modules
  • Eventing
  • Routing
  • Controllers
  • Directives
  • Services
  • Interceptors
  • Testability
  • Two way Model Binding
  • Extends Html
  • Makes Html more readable
