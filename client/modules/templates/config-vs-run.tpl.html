Only Providers and constants can be injected into config

angular.module('myModule').config(['AProvider', 'BProvider', ... , function(AProvider, BProvider, ...) { // Do Something }]);

Angular will apply constants first, then it applies the configs which is why we can inject constants as well.

Only instances and constants can be injected into run

angular.module('myModule').run(['AService', 'BService', ... , function(AService, BService, ...) { // Do Something }]);

Run Blocks get executed after the $injector service is created

You can have as many configs and runs as you'd like
