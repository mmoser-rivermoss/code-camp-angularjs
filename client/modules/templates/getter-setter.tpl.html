Create Module:
angular.module('riverMoss.modules', []);
Get Module:
// Notice there are no brackets angular.module('riverMoss.modules').controller(...);
Example of a module using dependencies
angular.module('riverMoss', ['ngCookies', 'ngResource', 'ui.bootstrap', 'ui.router', 'riverMoss.system', 'riverMoss.directives', 'riverMoss.modules', 'riverMoss.slides', 'hljs']);
